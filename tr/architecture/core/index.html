    Çekirdek Modüller¶

    Aspectify'in çekirdek modülleri, uygulamanın temel yapılandırma ve loglama işlevlerini sağlar.

    Config Modülü¶

    app/core/config.py dosyası, uygulamanın merkezi yapılandırma sistemini içerir.

    Settings Sınıfı¶

    from pydantic_settings import BaseSettings, SettingsConfigDict
from pydantic import SecretStr

class Settings(BaseSettings):
    model_config = SettingsConfigDict(env_file=".env", extra="ignore")

    PROJECT_NAME: str = "Aspectify"
    PROJECT_VERSION: str = "1.0.0"
    LOG_LEVEL: str = "INFO"
    GOOGLE_API_KEY: SecretStr | None = None
    GEMINI_1_5_FLASH_MODEL: str = "gemini-1.5-flash"

    Özellikler¶

    • Pydantic Tabanlı: Güçlü veri doğrulama
    • Ortam Değişkenleri: .env dosyası desteği
    • Gizli Veriler: API anahtarları için güvenli işleme
    • Varsayılan Değerler: Makul varsayılanlar
    • Tip Güvenliği: Statik tip kontrolü

    Kullanım¶

    from app.core.config import settings

# Temel ayarlar
print(f"Proje: {settings.PROJECT_NAME}")
print(f"Sürüm: {settings.PROJECT_VERSION}")

# Gizli değerler
if settings.GOOGLE_API_KEY:
    api_key = settings.GOOGLE_API_KEY.get_secret_value()

    Logging Config Modülü¶

    app/core/logging_config.py dosyası, uygulamanın loglama sistemini yapılandırır.

    Logging Yapılandırması¶

    import logging
import os
from app.core.config import settings

def setup_logging():
    log_level = settings.LOG_LEVEL.upper()

    # Logs dizini oluştur
    os.makedirs("logs", exist_ok=True)

    # Loglama yapılandırması
    logging.basicConfig(
        level=log_level,
        format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
        handlers=[
            logging.FileHandler('logs/app.log', encoding='utf-8')
        ]
    )

    logger = logging.getLogger(__name__)
    logger.info(f"Logging initialized with level: {log_level}")
    return logger

logger = setup_logging()

    Özellikler¶

    • Dosya Tabanlı: Loglar logs/app.log dosyasına yazılır
    • UTF-8 Desteği: Türkçe karakter desteği
    • Yapılandırılabilir Seviye: Ortam değişkeni ile kontrol
    • Otomatik Dizin: logs dizini otomatik oluşturulur
    • Timestamp: Her log girişi zaman damgası içerir

    Log Seviyeleri¶

    En İyi Uygulamalar¶

    Yapılandırma¶

    1. Varsayılan Değerler: Her ayar için makul varsayılanlar sağlayın
    2. Tip Doğrulama: Pydantic ile güçlü tip kontrolü
    3. Gizli Veriler: Hassas bilgileri SecretStr ile koruyun
    4. Dokümantasyon: Her ayar için açıklama ekleyin

    Loglama¶

    1. Uygun Seviye: Doğru log seviyesini kullanın
    2. Yapılandırılabilir: Ortam değişkeni ile kontrol edilebilir
    3. Performans: Loglama performansı etkilememeli
    4. Güvenlik: Hassas bilgileri loglamayın

    Hata Yönetimi¶

    1. Erken Kontrol: Yapılandırma hatalarını erken yakalayın
    2. Açıklayıcı Mesajlar: Hata mesajları açık ve anlaşılır olsun
    3. Graceful Degradation: Hata durumunda uygulama çökmesin
    4. Monitoring: Hataları izleyin ve raporlayın

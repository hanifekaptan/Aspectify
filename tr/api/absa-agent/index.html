    ABSA Agent API¶

    ABSA (Aspect-Based Sentiment Analysis) Agent, metin ve ses dosyalarında duygu analizi yapan ana bileşendir.

    Genel Bakış¶

    ABSA Agent, aşağıdaki işlemleri gerçekleştirir:

    1. İçerik Türü Belirleme: Girişin metin mi ses dosyası mı olduğunu tespit eder
    2. Ses Transkripsiyonu: Ses dosyalarını metne çevirir
    3. Dil Tespiti: Metnin dilini otomatik olarak belirler
    4. Duygu Analizi: Aspekt tabanlı duygu analizi yapar
    5. Sonuç Formatlama: Yapılandırılmış yanıt döndürür

    API Referansı¶

    run_absa Fonksiyonu¶

    async def run_absa(data: str) -> ABSAResponse

    Parametreler¶

    Parametre Tip Açıklama
    data str Analiz edilecek metin veya ses dosyası yolu

    Dönüş Değeri¶

    ABSAResponse nesnesi döndürür:

    class ABSAResponse(BaseModel):
    id: str                    # Benzersiz analiz ID'si
    polarity: str              # Genel duygu (positive/negative/neutral)
    score: float               # Güven skoru (0.0-1.0)
    aspects: List[AspectAnalysisResponse]  # Aspekt analizleri
    message: str               # İşlem durumu mesajı
    text: str                  # Analiz edilen metin

    Örnek Kullanım¶

    import asyncio
from app.agents.absa_agent import run_absa

async def main():
    # Metin analizi
    text = "Bu telefonun kamerası harika ama bataryası kötü."
    result = await run_absa(text)

    print(f"ID: {result.id}")
    print(f"Genel Duygu: {result.polarity}")
    print(f"Güven Skoru: {result.score}")
    print(f"Metin: {result.text}")

    for aspect in result.aspects:
        print(f"Aspekt: {aspect.aspect}")
        print(f"Duygu: {aspect.sentiment}")
        print(f"Skor: {aspect.score}")
        print(f"Alıntı: {aspect.quote}")

asyncio.run(main())

    İş Akışı¶

    1. İçerik Türü Belirleme¶

    from app.logic.content_type_identifier import identify_content_type

content_type = identify_content_type(data)
# "text" veya "audio_path" döndürür

    Desteklenen Ses Formatları: - .wav - WAV formatı - .mp3 - MP3 formatı - .m4a - M4A formatı - .flac - FLAC formatı

    2. Ses Transkripsiyonu¶

    from app.logic.transcriber import transcribe_from_path

if content_type == "audio_path":
    text_content = transcribe_from_path(data)

    Özellikler: - OpenAI Whisper modeli kullanır - Otomatik dil tespiti - Yüksek doğruluk oranı

    3. Dil Tespiti¶

    from app.logic.language_detector import detect_content_language

language = detect_content_language(text_content)
# "tr", "en", "de", vb. döndürür

    Desteklenen Diller: - Türkçe (tr) - İngilizce (en) - Almanca (de) - Fransızca (fr) - Ve daha fazlası...

    4. Duygu Analizi¶

    from app.tools.absa_analyzer import absa_analyzer_tool

result = absa_analyzer_tool.invoke({
    "text": text_content,
    "language": language
})

    Analiz Özellikleri: - Aspekt tabanlı analiz - Çoklu duygu tespiti - Güven skorları - Alıntı çıkarma

    Yanıt Modelleri¶

    ABSAResponse¶

    class ABSAResponse(BaseModel):
    id: str
    polarity: str
    score: float
    aspects: List[AspectAnalysisResponse]
    message: str
    text: str

    AspectAnalysisResponse¶

    class AspectAnalysisResponse(BaseModel):
    aspect: str        # Aspekt adı
    sentiment: str     # Duygu (positive/negative/neutral)
    score: float       # Güven skoru (0.0-1.0)
    quote: str         # Metinden alıntı

    Yaygın Hatalar¶

    Hata Açıklama Çözüm
    ValueError: The text content to be analyzed cannot be empty. Boş metin Geçerli metin sağlayın
    FileNotFoundError Ses dosyası bulunamadı Dosya yolunu kontrol edin
    API Key Error Geçersiz API anahtarı .env dosyasını kontrol edin
    Model Error AI model hatası Model yapılandırmasını kontrol edin

    Hata Yakalama¶

    try:
    result = await run_absa(data)
    print(f"Analiz başarılı: {result.polarity}")
except ValueError as e:
    print(f"Geçersiz giriş: {e}")
except FileNotFoundError as e:
    print(f"Dosya bulunamadı: {e}")
except Exception as e:
    print(f"Beklenmeyen hata: {e}")

    Performans¶

    Optimizasyon İpuçları¶

    1. Asenkron Kullanım: async/await kullanın
    2. Toplu İşlem: Birden fazla metni aynı anda işleyin
    3. Önbellekleme: Sonuçları önbelleğe alın
    4. Dosya Boyutu: Büyük ses dosyalarını önceden işleyin

    Performans Metrikleri¶

    • Metin Analizi: ~2-5 saniye
    • Ses Transkripsiyonu: ~10-30 saniye (dosya boyutuna göre)
    • Dil Tespiti: ~0.1-0.5 saniye
    • Toplam Süre: ~15-35 saniye

    Unit Testler¶

    import pytest
from app.agents.absa_agent import run_absa

@pytest.mark.asyncio
async def test_run_absa_with_text():
    text = "Bu ürün harika!"
    result = await run_absa(text)

    assert result.polarity in ["positive", "negative", "neutral"]
    assert 0.0 <= result.score <= 1.0
    assert len(result.aspects) > 0
    assert result.text == text

@pytest.mark.asyncio
async def test_run_absa_with_audio():
    audio_path = "test_data/audio/sample.wav"
    result = await run_absa(audio_path)

    assert result.polarity in ["positive", "negative", "neutral"]
    assert len(result.text) > 0

    En İyi Uygulamalar¶

    Kullanım¶

    1. Asenkron Kullanım: Her zaman async/await kullanın
    2. Hata Yakalama: Try-catch blokları kullanın
    3. Doğrulama: Giriş verilerini doğrulayın
    4. Loglama: İşlemleri loglayın

    Performans¶

    1. Toplu İşlem: Birden fazla metni aynı anda işleyin
    2. Önbellekleme: Sonuçları önbelleğe alın
    3. Kaynak Yönetimi: Bellek kullanımını izleyin
    4. Timeout: Uzun süren işlemler için timeout ayarlayın

    Güvenlik¶

    1. Input Validation: Giriş verilerini doğrulayın
    2. File Security: Dosya yollarını kontrol edin
    3. API Keys: API anahtarlarını güvenli tutun
    4. Error Messages: Hassas bilgileri hata mesajlarında göstermeyin

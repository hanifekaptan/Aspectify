    Sistem Gereksinimleri¶

    Aspectify'in çalışması için gerekli sistem gereksinimleri ve bağımlılıkları.

    Minimum Sistem Gereksinimleri¶

    İşletim Sistemi¶

    • Windows: Windows 10 veya üzeri
    • macOS: macOS 10.15 (Catalina) veya üzeri
    • Linux: Ubuntu 18.04 LTS veya üzeri

    Donanım¶

    Bileşen Minimum Önerilen
    CPU 2 çekirdek, 2.0 GHz 4 çekirdek, 3.0 GHz
    RAM 4 GB 8 GB
    Disk 2 GB boş alan 10 GB boş alan
    Ağ İnternet bağlantısı Stabil internet bağlantısı

    Yazılım Gereksinimleri¶

    Python¶

    • Python: 3.8 veya üzeri
    • pip: 20.0 veya üzeri

    Python Sürümü Kontrolü¶

    python --version
# Python 3.8.0 veya üzeri olmalı

pip --version
# pip 20.0 veya üzeri olmalı

    Kurulum¶

    1. Python Sanal Ortamı¶

    # Sanal ortam oluştur
python -m venv .venv

# Sanal ortamı aktifleştir
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

    2. Bağımlılıkları Yükle¶

    # Tüm bağımlılıkları yükle
pip install -r requirements.txt

# Veya adım adım
pip install fastapi pydantic uvicorn
pip install langchain langchain-google-genai
pip install openai-whisper torch
pip install databases[postgresql] psycopg2-binary
pip install pytest mkdocs-material

    3. Sistem Bağımlılıkları¶

    Windows¶

    # Visual C++ Build Tools (torch için)
# https://visualstudio.microsoft.com/visual-cpp-build-tools/

# FFmpeg (Whisper için)
# https://ffmpeg.org/download.html

    macOS¶

    # Homebrew ile
brew install ffmpeg

# Xcode Command Line Tools
xcode-select --install

    Linux (Ubuntu/Debian)¶

    # Sistem paketlerini güncelle
sudo apt update

# Gerekli paketleri yükle
sudo apt install -y python3-dev python3-pip
sudo apt install -y ffmpeg
sudo apt install -y build-essential
sudo apt install -y libsndfile1

    API Anahtarları¶

    Google API Anahtarı¶

    1. Google AI Studio adresini ziyaret edin
    2. Yeni bir proje oluşturun
    3. API anahtarı oluşturun
    4. .env dosyasına ekleyin:
    GOOGLE_API_KEY=your_google_api_key_here

    Pinecone API Anahtarı (İsteğe Bağlı)¶

    1. Pinecone Console adresini ziyaret edin
    2. Hesap oluşturun
    3. API anahtarı alın
    4. .env dosyasına ekleyin:
    PINECONE_API_KEY=your_pinecone_api_key
PINECONE_ENVIRONMENT=your_environment
PINECONE_INDEX_NAME=aspectify-index

    Veritabanı Gereksinimleri¶

    PostgreSQL¶

    • Sürüm: 12 veya üzeri
    • Port: 5432 (varsayılan)
    • Kullanıcı: aspectify
    • Şifre: güçlü şifre
    • Veritabanı: aspectify_db

    Kurulum¶

    Windows¶

    # PostgreSQL indir ve yükle
# https://www.postgresql.org/download/windows/

    macOS¶

    # Homebrew ile
brew install postgresql
brew services start postgresql

    Linux¶

    # Ubuntu/Debian
sudo apt install postgresql postgresql-contrib
sudo systemctl start postgresql
sudo systemctl enable postgresql

    Veritabanı Oluşturma¶

    -- PostgreSQL'e bağlan
psql -U postgres

-- Kullanıcı oluştur
CREATE USER aspectify WITH PASSWORD 'your_password';

-- Veritabanı oluştur
CREATE DATABASE aspectify_db OWNER aspectify;

-- İzinleri ver
GRANT ALL PRIVILEGES ON DATABASE aspectify_db TO aspectify;

    Performans Gereksinimleri¶

    CPU Kullanımı¶

    • Metin Analizi: %20-40 CPU kullanımı
    • Ses Transkripsiyonu: %60-80 CPU kullanımı
    • Toplu İşlem: %80-100 CPU kullanımı

    Bellek Kullanımı¶

    • Temel Çalışma: 2-4 GB RAM
    • Ses İşleme: 4-8 GB RAM
    • Toplu Analiz: 8-16 GB RAM

    Disk Kullanımı¶

    • Uygulama: 2 GB
    • Model Dosyaları: 3-5 GB
    • Log Dosyaları: 100-500 MB
    • Geçici Dosyalar: 1-2 GB

    Ağ Gereksinimleri¶

    Gerekli Portlar¶

    Port Protokol Açıklama
    8000 HTTP FastAPI uygulaması
    5432 TCP PostgreSQL
    443 HTTPS API çağrıları

    Dış Servisler¶

    • Google AI API: generativelanguage.googleapis.com
    • Pinecone API: api.pinecone.io
    • PyPI: pypi.org (paket yükleme)

    Güvenlik Gereksinimleri¶

    SSL/TLS¶

    • HTTPS: Üretim ortamında zorunlu
    • Sertifika: Geçerli SSL sertifikası
    • Şifreleme: TLS 1.2 veya üzeri

    Güvenlik Duvarı¶

    # Gerekli portları aç
sudo ufw allow 8000/tcp
sudo ufw allow 5432/tcp
sudo ufw enable

    Kullanıcı İzinleri¶

    # Uygulama kullanıcısı oluştur
sudo useradd -m -s /bin/bash aspectify
sudo usermod -aG docker aspectify

# Dizin izinleri
sudo chown -R aspectify:aspectify /opt/aspectify
sudo chmod -R 755 /opt/aspectify

    Sorun Giderme¶

    Yaygın Sorunlar¶

    1. Python Sürüm Uyumsuzluğu¶

    # Python sürümünü kontrol et
python --version

# Gerekirse Python 3.8+ yükle
# Windows: https://www.python.org/downloads/
# macOS: brew install python@3.8
# Linux: sudo apt install python3.8

    2. Bağımlılık Çakışması¶

    # Sanal ortamı temizle
deactivate
rm -rf .venv
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

    3. API Anahtarı Hatası¶

    # .env dosyasını kontrol et
cat .env

# API anahtarını doğrula
curl -H "Authorization: Bearer $GOOGLE_API_KEY" \
     https://generativelanguage.googleapis.com/v1/models

    4. Veritabanı Bağlantı Hatası¶

    # PostgreSQL durumunu kontrol et
sudo systemctl status postgresql

# Bağlantıyı test et
psql -h localhost -U aspectify -d aspectify_db

    Log Dosyaları¶

    # Uygulama logları
tail -f logs/app.log

# Sistem logları
sudo journalctl -u aspectify -f

# Veritabanı logları
sudo tail -f /var/log/postgresql/postgresql-*.log

    En İyi Uygulamalar¶

    Kurulum¶

    1. Sanal Ortam: Her zaman sanal ortam kullanın
    2. Sürüm Sabitleme: requirements.txt'de sürümleri sabitleyin
    3. Güvenlik: API anahtarlarını güvenli tutun
    4. Backup: Veritabanını düzenli yedekleyin

    Performans¶

    1. Kaynak İzleme: CPU, RAM, disk kullanımını izleyin
    2. Önbellekleme: Sonuçları önbelleğe alın
    3. Load Balancing: Yüksek trafik için load balancer kullanın
    4. Monitoring: Sistem metriklerini izleyin

    Güvenlik¶

    1. Güncellemeler: Sistem ve paketleri güncel tutun
    2. Güvenlik Duvarı: Gereksiz portları kapatın
    3. SSL: HTTPS kullanın
    4. Backup: Düzenli yedekleme yapın
